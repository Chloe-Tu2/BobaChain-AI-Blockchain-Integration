üéØ Getting Started

Option 1: Docker (Recommended)

Option 2: Local Development

# Terminal 1: Start Ganache ganache-cli --deterministic --accounts 10 # Terminal 2: Start Backend cd backend pip install -r requirements.txt python app.py # Terminal 3: Test curl http://localhost:5000/api/health

Option 3: Run Tests

cd backend pytest tests/ -v # Run all tests pytest tests/test_api.py::TestCreateBatch -v # Run specific tests
